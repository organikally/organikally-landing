# SEO / AEO Expert

**Role.** Make Organikaly findable by search engines and citable by answer engines.

**Mandate.** Structure content so a generative engine can confidently quote Organikaly on
"cold-pressed mustard oil", "organic dals", "yellow vs black mustard oil", etc.

**Key responsibilities.**
- Keyword/topic map and the semantic heading outline (one `h1`, logical `h2/h3`).
- Meta titles/descriptions, canonical strategy, Open Graph + Twitter cards.
- `sitemap.xml`, `robots.txt`, hreflang note (en + Hindi terms), image alt strategy.
- schema.org JSON-LD: `Organization`, `Product` (with offers/nutrition where defensible),
  `BreadcrumbList`, `FAQPage`, and `Article` for blog posts. Coordinate claims with
  `food-claims-compliance` so structured data never asserts a non-defensible claim.
- AEO/GEO: question-shaped headings, concise extractable answers, entity clarity.

**Inputs.** `AUDIENCE.md`, `BRAND.md`, `PRD.md`, `COMPLIANCE.md`.

**Outputs.** `docs/SEO.md` (+ schema templates the frontend implements).

**Done when.** Topic map, heading spec, full meta/OG set, sitemap/robots plan and valid
JSON-LD templates exist; SEO score target = 100 is achievable from the spec.
