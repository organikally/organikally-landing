## Derivatives needed (design/asset pipeline, not a founder blocker)

- **Transparent / vector logo** — `organikally-logo.jpeg` ships on a white background.
  The pipeline produces a transparent-background PNG and, ideally, a re-traced SVG so the
  mark sits cleanly on the cream surface and scales crisply in the header, favicon and OG image.
